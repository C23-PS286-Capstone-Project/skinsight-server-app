# Server for SkinSight App

Backend apps for SkinSight

## Setup

Note : Required NodeJS version >= 18.0.0

- Check npm version

```bash
npm -v
```

- Install dependencies

```bash
npm install
```

- Copy .env.example to .env

```bash
cp .env.example .env
```
